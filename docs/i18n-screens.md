# Экраны админ‑панели для проверки i18n

Список ключевых экранов для проверки полноты переводов (ru, en, he) и корректности RTL.

| Маршрут | Страница | Статус |
|--------|----------|--------|
| / | Главная (дашборд) | ✅ переведено |
| /auth | Вход / Регистрация | ✅ переведено |
| /programs | Программы (список, создание, редактирование, удаление) | ✅ переведено |
| /categories | Категории (список, создание, редактирование, удаление) | ✅ переведено |
| /skills | Навыки (список, создание, редактирование) | ✅ переведено |
| /tasks | Таски (список, создание, назначение волонтёра) | ✅ переведено |
| /users | Пользователи (список, детали, назначение программ) | ✅ переведено |
| /cities | Города (список, создание, волонтёры по городу) | ✅ переведено |
| /city-groups | Группы городов (список, создание, редактирование, управление городами) | ✅ переведено |
| * (404) | Страница «Не найдена» | ✅ переведено |

**Проверка**: для каждого экрана переключить язык на ru → en → he и убедиться, что нет смешения языков и технических ключей; для he проверить RTL.
