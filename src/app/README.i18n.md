# I18n в админ‑панели

Файлы `src/app/` отвечают за инициализацию приложения и подключение провайдеров.

- Точка входа: `src/app/index.tsx`.
- Провайдер `I18nProvider` из `src/shared/lib/i18n` оборачивает всё приложение и:
  - хранит текущий язык (`ru`, `en`, `he`);
  - обеспечивает доступ к функции перевода `t(key, params?)`;
  - обновляет атрибуты `dir` и `lang` на элементе `<html>` при смене языка.

Компоненты и страницы должны получать тексты через хук `useI18n` и функцию `t`, а не использовать «сырые» строки в JSX.
